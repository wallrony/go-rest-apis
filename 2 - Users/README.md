# Users GO Project

Esse é um projeto de uma API Rest somente com um CRUD simples com o objeto
User, manipulado somente na aplicação, sem nenhuma api de persistência de dados 
feita em GO, utilizando a request api Mux para gerenciar as rotas de requisição
da aplicação.

Em breve terá um detalhamento melhor sobre as rotas da aplicação e seus recursos.
